# Student Utility Program - Web Version

This is a web-based version of the Student Utility Program, originally written in C. It provides various calculators and tools for students.

## Features

1. **Percentage Calculator** - Calculate percentage from obtained and total marks
2. **CGPA Calculator** - Calculate CGPA from grade points and credit hours
3. **Fibonacci Generator** - Generate Fibonacci sequence up to n terms
4. **Factorial Calculator** - Calculate factorial of a number
5. **Unit Converter** - Convert between temperature, length, and weight units
6. **Physics Calculator** - Solve common physics formulas (Kinematics, Dynamics, Energy, Electricity)
7. **Number Tools** - Check if a number is prime, even/odd, or reverse it

## How to Run

1. Open the `index.html` file in any modern web browser
2. The main menu will be displayed
3. Click on any option to access that calculator/tool
4. Enter the required inputs and click calculate
5. Use the "Back to Menu" button to return to the main menu

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling
- `script.js` - JavaScript functionality

## Browser Compatibility

This web application works in all modern browsers that support HTML5, CSS3, and ES6 JavaScript.

## Note

This is a conversion of the original C program to web technologies. All calculations are performed client-side in the browser.
